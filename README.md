# Shimoji-eeAnimator
A easy way to animate Playgrounds in Shimoji-ee

This is a application to convert a .mp4 into frames

more readme info coming soon, this is a early beta for apps! and still in development!

# Installation

Download the .py file and double click it!
(to run this file you need PySide6)

# PySide6
use ```pip install PySide6``` to install it!
